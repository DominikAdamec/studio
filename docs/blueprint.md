# **App Name**: Prompty

## Core Features:

- Prompt Ideas: Generate imaginative and unique prompt ideas for AI image generation, allowing users to explore diverse creative directions.
- Prompt Library: Enable users to save their favorite prompts and organize them into collections for quick access and reuse, making prompt management effortless.
- Copy Prompts: Facilitate quick copying of prompts to the clipboard for immediate use in AI image generators, streamlining the creative workflow.
- Describe photo for Flux1.Dev: Describe the contents of a photo and generate prompts for the Flux1.Dev AI image generator. The LLM will use its tool to decide when to incorporate information in its output.
- Subscription System: Enable users to subscribe to 'Prompty PRO' to unlock advanced features.
- Art Style Prompts: PRO: Generate prompts based on a specific artistic style (e.g., Van Gogh, Impressionism).
- Prompt Refinement: PRO: Refine existing prompts to be more detailed and specific, enhancing the quality of generated images.
- Prompt Suggestions: PRO: Suggest related prompts based on user's saved or recently used prompts.
- Advanced Organization: PRO: Enhanced organization features with nested collections and tagging.
- Export Prompts: PRO: Export prompts in various formats (JSON, CSV, text) for use in other applications.
- AI Chat Master: PRO: AI chat master can talk with you about photo which you upload to chat, can help you to edit prompts which user send to chat, user can describe only parts which user need (example, user ask you to describe only shoes from character, or nature at background and etc..)

## Style Guidelines:

- Implement a dark background to reduce eye strain and provide a sleek, modern look. (Tailwindcss dark mode)
- Dark purple (#4E56A8) to set the mood and add a bit of elegance. (Tailwindcss remix)
- Teal (#00B8A9) to provide vibrant highlights, creating visual contrast on the dark background. (Tailwindcss remix)
- 'Inter' sans-serif for clean and modern readability.
- Use simple, minimalist icons in teal to complement the dark theme and provide clear visual cues.
- Subtle transition animations to create a smooth and engaging user experience.
- Follow iOS design principles for a native look and feel.
